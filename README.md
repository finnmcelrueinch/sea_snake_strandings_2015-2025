# sea_snake_strandings_2015-2025
This repository contains code used in my BSc dissertation: 'ASSESSING PATTERNS AND DRIVERS OF SEA SNAKE (SUBFAMILY HYDROPHIINAE) STRANDINGS AROUND AUSTRALIA FROM 2015 - 2025.'

Workflows include quality assurance/control measures, spatial analyses, temporal analyses, veterinary analyses, and the modelling of environmental variables. Please contact s2345947@ed.ac.uk if you require further assistance. To run please install the latest version of R and the required packages outlined in each workflow.
